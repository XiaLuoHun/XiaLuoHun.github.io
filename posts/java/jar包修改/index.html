Jar包修改

 夏洛魂 收录于 Java
2021-10-01 约 300 字 预计阅读 1 分钟 
  1. 将目标jar包拷贝到IDEA工程的lib目录下,然后右键lib目录,Add as Library.

  1. 在IDEA中右键Src目录,新建一个Java类
1
2
3

如:
目标类: com.android.xxx.a.class
新建类: com.android.xxx.a.java
  1. 用Jeb反编译目标class文件,拷贝到新建类中.
  2. 在新建类中进行修改.
  3. IDEA编译修改后的工程,找到编译出来的.class文件.
  4. 将Jar包用压缩软件解压,用编译出来的.class文件进行替换.
  5. cd到Jar包解压文件所在目录,利用jdk bin目录中的jar进行重新打包.
1
2

jar cvf XXX.jar *
//XXX.jar为要打包成的jar文件名
  1. 用新生成的Jar包替换之前的Jar即可.
