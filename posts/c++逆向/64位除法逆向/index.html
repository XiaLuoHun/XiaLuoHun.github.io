x / 除数 =  x * M  >> (32 + n)
结论:
M > 0: x / 正除数, 除数 = 2 ^ (32 + n) / M, 向上取整
M < 0: x / 负除数, 除数 = 2 ^ (32 + n) / (2 ^ 32 - M ), 向上取整

x / 除数 =  (x * M + x) >> (32 + n)
结论:
M < 0: x / 正除数, 除数 = 2 ^ (32 + n) / M, 向上取整

x / 除数 =  (x * M - x) >> (32 + n)
结论:
M > 0: x / 负除数, 除数 = 2 ^ (32 + n) / (2 ^ 32 - M ), 向上取整
