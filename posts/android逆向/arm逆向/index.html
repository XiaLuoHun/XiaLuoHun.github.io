上述特征为:x / y = x * c >> 32 >> n
还原的话就看c,分为3种情况:
c为正数:y = 2^n / c
c为负数:y = 2^n / (2^32 - c)
c溢出:  y = 2^n / (2^32 + c)
以上述arm-v7a为例:
c = 0x38E38E39 = 954437177, 为正.
n = 32 + 1 = 33
则 y = 2^33 / 954437177 = 9
